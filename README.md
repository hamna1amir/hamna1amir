</div>

An AI-powered agentic system that helps users analyze stock options through natural language — combining LLM reasoning, RAG, and specialized financial tools to retrieve, analyze, and visualize real-time market data.

| Layer | Technology |
|---|---|
| Orchestration | LangGraph (22 tools) |
| Market Data | Polygon.io (live options data) |
| Knowledge Retrieval | ChromaDB (RAG) |
| Memory | SQLite (persistent conversation memory) |
| Reporting | Chart & CSV generation |
| Deployment | FastAPI microservice |
| Extras | Web search, performance monitoring, evaluation pipelines |
